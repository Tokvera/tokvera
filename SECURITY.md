# Security Policy

## Reporting a Vulnerability

If you discover a security issue, report it privately to:

`security@tokvera.com`

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested mitigation

## Responsible Disclosure

Please do not publicly disclose vulnerabilities before maintainers have had a reasonable opportunity to investigate and remediate.  
We will acknowledge valid reports and work to resolve issues in a timely manner.
